# BLOG REST API

Rest Api that allows perform basic operation that a simple Blog app needs.

## Spring Modules and Dependencies

* Spring BOOT
* Spring WEB
* Spring JDBC
* MySQL JDBC

## Prerequisites

* Java 8
* Gradle 4.4.1

## Launch REST API

```
$ ./gradlew bootRun
```

## Running the tests

```
$ ./gradlew test
```
